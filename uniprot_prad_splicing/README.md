# UniProt: alternative splicing of prostate-cancer-related proteins

Generated by `analysis/uniprot_prad_splicing.py` from the UniProtKB REST API.

## Query

```
(organism_id:9606) AND (reviewed:true) AND (keyword:KW-0025) AND ("prostate cancer")
```

i.e. human, reviewed (Swiss-Prot) proteins carrying the **Alternative splicing**
keyword (KW-0025) whose entry mentions prostate cancer. The exact query, fields,
and record count are recorded in `query.txt`.

## Results (this run)

- **239** proteins matched; all have >1 curated isoform.
- **12** carry a curated *prostate-cancer* disease annotation — including the
  hereditary-PCa loci ELAC2 (HPC2), RNASEL (HPC1), MSR1, EHBP1, MSMB, CHEK2,
  plus AR, PTEN, EPHB2, HNF1B, MXI1, KLF6.
- **871** isoform sequences across the 239 proteins.

## Files

| File | Contents |
|------|----------|
| `proteins.tsv` | One row per protein: accession, gene, name, length, #isoforms, #splice-variant features, whether it has a prostate-cancer disease term, disease list, isoform IDs/names, function. |
| `proteins.json` | Full UniProt records (alternative-products comments, VAR_SEQ features, disease, function, sequence). |
| `isoforms.fasta` | FASTA of every annotated isoform (canonical + `-N` isoforms). |
| `query.txt` | Provenance: endpoint, query, fields, record count. |

## Notes

- `includeIsoform=true` on a keyword/organism query drops isoforms (they carry
  no such fields), so isoforms are re-fetched by accession in batches.
- Reproduce: `python analysis/uniprot_prad_splicing.py`
  (custom search: `--query '<UniProt query>'`).
- This is a curated-knowledge snapshot from UniProt; it complements the
  cohort-derived events in `results/` and the report in `docs/`.
